## 快速开始

### 对于新用户
1. **部署服务器**（选择以下任一方式）：
   - **🐋 Docker 方式（最推荐）**:
     ```bash
     cd server-native && chmod +x deploy.sh && ./deploy.sh
     ```
     详见 **[Docker部署指南](../server-native/README-DOCKER.md)**
   - **🐧 Linux 传统方式**: 按照 **[Linux服务器部署指南.md](./Linux服务器部署指南.md)** 部署
   - **🪟 Windows 方式**: 运行 `server-native/1-install.bat` 部署（不推荐，存在稳定性问题）
2. **配置应用**：按照 **[配置和使用指南.md](./配置和使用指南.md)** 配置数据库连接
3. **编译安装**：运行 `gradlew.bat assembleDebug` 编译并安装到设备
4. 阅读 **[项目概述.md](./项目概述.md)** 了解项目整体功能

## 项目特色

小光AI助手是一个采用**心流驱动架构**的智能AI助手，具有以下特色：

- 🧠 **心流驱动决策**: 每3秒持续运行的主动决策系统
- 🎨 **情绪驱动UI**: 界面颜色和风格随小光情绪实时变化
- 🗂️ **智能记忆管理**: 基于知识图谱的多维度记忆系统
- 👥 **社交关系感知**: 自动提取和更新人物关系
- 🎤 **语音交互支持**: 语音唤醒和离线语音识别
- 🔊 **声纹识别系统**: 自动识别说话人身份，智能管理人物档案
- 💝 **主动关怀提醒**: 基于环境和用户状态的智能提醒
- ✨ **流畅动画体验**: 丰富的页面转场和交互动画

## 技术栈

### 前端技术
- **UI框架**: Jetpack Compose + Material3
- **架构模式**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **状态管理**: StateFlow + Compose State
- **动画系统**: Compose Animation API

### 后端技术
- **开发语言**: Kotlin
- **数据库**: Realm + ChromaDB + Neo4j
- **AI模型**: 硅基流动API
- **语音技术**: Porcupine + Vosk + ONNX Runtime
- **网络请求**: Retrofit + OkHttp
